User Manual

Tools


Pan

To use the pan tool click and drag screen and the canvas will move, alternately press the space bar on any tool(as long as you arent currently drawing) and darg around to move the canvas

Shape

to place a shape click and drag, size to needs and then release the mouse to create the shape


A variety of options available such as the stroke thickness, shape, and fill in. click either of these to use it

Line

Click and drag the mouse and it draws a line in real time


Options available include stroke thickness

Text

After clicking, you can type out text and it will appear as an object, note new lines are not supported


Options available include font size

Object

To select an object simply click it, to move an object click and drag it, and to delete an object press the DEL key

Colour


The colour window gives you a preview of the current colour, and the ability to edit RGB values via sliders, the value updates on the main window
